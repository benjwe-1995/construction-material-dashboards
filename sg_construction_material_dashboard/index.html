Singapore Construction Material Price Dashboard

Interactive dashboard using two public data.gov.sg datasets: BCA construction material market prices and SingStat import price indices by commodity group. The server fetches and transforms the data; the browser only renders the charts.

Source: data.gov.sg datastore API

Dataset 1 — Construction Material Market Prices

Raw monthly market-price series for Cement, Steel Rebar, Granite Aggregate, Sand, and Ready-mixed Concrete.

Dataset 2 — Import Price Indices

Monthly import price index series for Crude Oil, Copper, Aluminium, and Steel. Base year: 2023 = 100.

Custom Period % Change

Select a start and end date. The calculator uses the closest available month in each loaded series.
